# Tic-Tac-Toe-Game
A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. Play against a friend in this classic two-player game — the first to get three in a row wins!

# Features:

Smooth and responsive UI.

Player turn indication (X and O).

Winner detection and game-over alert.

Reset button to start a new match instantly.

Simple, clean, and fun design.

# Built With:

HTML – Structure of the game board.

CSS – Styling with hover and color effects.

JavaScript – Game logic, win checking, and interactivity.
